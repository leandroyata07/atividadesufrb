# Currículo Vitae Interativo - Leandro Yata 📄✨

Este é um projeto acadêmico de desenvolvimento web voltado para a prática de HTML5 semântico, CSS3 responsivo e manipulação dinâmica do DOM (Document Object Model) com JavaScript puro (Vanilla JS).

O projeto consiste em um mini-currículo online responsivo contendo informações acadêmicas, habilidades, dados de contato e recursos interativos.

---

## 🚀 Funcionalidades

### 1. Estruturação Semântica e Acessibilidade
*   Desenvolvido utilizando tags HTML5 semânticas apropriadas (`<header>`, `<nav>`, `<main>`, `<section>`, `<aside>`, `<address>`, `<footer>`).
*   Navegação acessível estruturada com listas e links adequados.

### 2. Design Moderno e Responsivo
*   Visual construído com variáveis CSS para fácil manutenção e customização de temas.
*   Layout responsivo adaptável a dispositivos móveis (smartphones, tablets) e desktops.
*   Efeitos dinâmicos de *hover* e transições suaves nos cards e botões.

### 3. Interatividade com o DOM (JavaScript)
*   **Alternador de Tema (Modo Claro/Escuro):** Botão interativo no cabeçalho de todas as páginas que altera o esquema de cores global do site. A preferência do usuário é salva no `localStorage` do navegador para persistência ao trocar de página.
*   **Adição Dinâmica de Habilidades:** Na aba de Habilidades, o usuário pode digitar uma nova habilidade e adicioná-la dinamicamente na lista atual por meio de cliques no botão ou pressionando a tecla `Enter`.
*   **Feedback em Tempo Real (Input Event):** Enquanto o usuário digita uma habilidade, um indicador exibe em tempo real a quantidade de caracteres inseridos.
*   **Destaque Dinâmico de Itens:** Ao clicar em qualquer habilidade da lista (existente ou recém-criada), o item altera seu estilo visual, destacando-se dos demais.

## 📁 Estrutura de Arquivos

*   `index.html` — Página inicial de boas-vindas ao currículo.
*   `sobre.html` — Detalhes sobre o autor e link para download de versão PDF.
*   `educacao.html` — Informações sobre a trajetória acadêmica na UFRB.
*   `habilidades.html` — Lista de habilidades técnicas e seção de manipulação dinâmica do DOM.
*   `contato.html` — Canais de comunicação (e-mail e GitHub).
*   `styles.css` — Estilização moderna, variáveis CSS, responsividade e estilos de temas.
*   `scripts.js` — Lógica em JavaScript puro para captura de eventos e alteração de estilo e conteúdo no DOM.

---

## 🛠️ Como Executar o Projeto Localmente

1. Faça o clone ou baixe este repositório em seu computador.
2. Navegue até a pasta do projeto.
3. Dê um duplo clique no arquivo `index.html` para abri-lo diretamente no seu navegador de preferência, ou utilize uma extensão como a *Live Server* no VS Code.
